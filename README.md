# recipebookapp


## 🚀 Key Features

- 🍽️ Browse recipes by categories (Breakfast, Lunch, Dinner)
- 🔍 Search recipes by name
- 📖 View detailed recipe information
- 🥕 Display ingredients list for each recipe
- ❤️ Save favorite recipes for quick access
- 🔄 State management using Provider
- ⚡ Optimized API calls with reusable NetworkCaller
- 📱 Clean and responsive UI design

  ## 🛠️ Tech Stack

- Flutter
- Dart
- REST API (Spoonacular API)
- Provider (State Management)
- HTTP Networking
